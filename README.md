# Full Stack Task Manager

A beginner-friendly full stack web application that allows users to add, view, and delete tasks.

## Tech Stack
- Frontend: HTML, CSS, JavaScript
- Backend: Node.js, Express.js

## Features
- Add new tasks
- View task list
- Delete tasks
- REST API integration
